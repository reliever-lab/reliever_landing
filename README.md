# Reliever 랜딩페이지

이 저장소는 기업 코드베이스 인텔리전스를 위한 슬랙 기반 에이전트를 배포하는 AX 기업 Reliever의 Vercel 배포용 Next.js 랜딩페이지입니다.

## 기술 스택

- Next.js App Router
- React
- TypeScript
- Reliever 메인 컬러 `#0E4274` 기반의 순수 CSS

## 로컬 개발

```bash
npm install
npm run dev
```

브라우저에서 `http://localhost:3000`을 엽니다.

## 프로덕션 빌드

```bash
npm run build
```

기본 Next.js 프레임워크 프리셋으로 Vercel에 배포할 수 있습니다.
